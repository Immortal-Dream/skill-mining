# Astronomy Dataset Bindings

Canonical benchmark files:

- `benchmark/datasets/astronomy/m67_star_image.fits`
- `benchmark/datasets/astronomy/spitzer_irac_45_image.fits`
- `benchmark/datasets/astronomy/spitzer_irac_45_catalog.xml`
- `benchmark/datasets/astronomy/4gaussians_params.ecsv`
- `benchmark/datasets/astronomy/100gaussians_params.ecsv`

Status:

- All listed files are `ready`.

Intended use:

- M67: small real-image background and detection tasks.
- Spitzer image plus catalog: small real-image photometry and catalog-grounded tasks.
- Photutils bundled parameter tables: deterministic synthetic-image generation tasks.
