# Biology Dataset Bindings

Canonical benchmark files:

- `benchmark/datasets/biology/pbmc3k_raw.h5ad`
- `benchmark/datasets/biology/pbmc68k_reduced.h5ad`

Status:

- Both are `ready`.

Intended use:

- `pbmc3k_raw.h5ad`: raw-ish Scanpy PBMC input for scPSS scoring and significance tasks.
- `pbmc68k_reduced.h5ad`: small processed PBMC input for Harmony comparison and parameter tasks.
